Проект "Анализ посещений сайта "СберАвтоподписка"

Исходные данные:
Датасеты:
Данные о посещении сайта за 2021 год,
Данные о действиях посетителя в рамках одного посещения


Цель: 
Обучить модель для предсказания совершения пользователем 
целевого действия с метрикой ROC-AUC не менее 0,65


Основные этапы проекта:
Чтение датасета
Очистка данных
Генерация новых признаков
Разведочный анализ
Обучение ML-моделей и оценка их эффективности
Построение пайплана
Реализация API - сервиса на основе лучшей модели.
